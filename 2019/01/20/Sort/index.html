七大排序算法

  • 2019-01-20

一. 冒泡排序

1.算法描述
     依次比较两个相邻元素，如果前面一个数比后面一个数大，就交换两者的位置，每次循环以此找出最大的数。重复以上操作。时间复杂度为O(n^2)，空间复杂度O(1)，算法稳定。
2.算法实现

   public static void Bubble_Sort(int []array){
    for (int i = 0; i < array.length-1; i++) {
        for (int j = 0; j < array.length-1-i; j++) {
            if (array[j]>array[j+1]){
                int tmp = array[j];
                array[j] = array[j+1];
                array[j+1] = tmp;
            }
        }
    }
}

2.选择排序

1.算法描述
     在未排序序列中找到最小元素，存放到排序序列的起始位置，然后，再从剩余未排序元素中继续寻找最小元素，然后放到已排序序列的末尾。时间复杂度为O(n^2)，空间复杂度O(1)，算法不稳定。
2.算法实现

public static void Selection_Sort(int []array){
    for (int i = 0; i < array.length-1; i++) {
        int minindex = i;
        for (int j = i + 1; j < array.length; j++) { //7
            if (array[j]<array[minindex]){
                minindex = j;
            }
        }
        int tmp = array[minindex];
        array[minindex] = array[i];
        array[i] = tmp;
    }
}

3.插入排序

1.算法描述
     对于未排序数据，在已排序序列中从后向前扫描，找到相应位置并插入。时间复杂度为O(n^2)，空间复杂度O(1)，算法稳定。
2.算法实现

public  static void Insert_Sort(int []array){
    int currrent;
    for (int i=0;i<array.length-1;i++){
        currrent=array[i+1];
        int preIndex=i;
        while (preIndex>=0 && currrent<array[preIndex]){
            array[preIndex+1]=array[preIndex];
            preIndex--;
        }
        array[preIndex+1]=currrent;
    }
}

4.希尔排序

1.算法描述
     是简单插入排序的改进版。它与插入排序的不同之处在于，它会优先比较距离较远的元素。希尔排序又叫缩小增量排序。时间复杂度为小于O(n^2)，空间复杂度O(1)，算法不稳定。
2.算法实现

public static void Shell_Sort(int[] array){
    int n = array.length;
    int gap = n/2;
    while(gap > 0){
        for(int j = gap; j < n; j++){
            int index=j;
            while(index >= gap && array[index-gap] > array[index]){
                int temp = array[index-gap];
                array[index-gap] = array[index];
                array[index] = temp;
                index -= gap;
            }
        }
        gap = gap/2;
    }
}

5.归并排序

1.算法描述
     归并排序是建立在归并操作上的一种有效的排序算法。该算法是采用分治法（Divide and Conquer）的一个非常典型的应用。将已有序的子序列合并，得到完全有序的序列；即先使每个子序列有序，再使子序列段间有序。若将两个有序表合并成一个有序表。时间复杂度为小于O(n^2)，空间复杂度O(n，算法稳定。
2.算法实现

vate static void sort(int[] arr,int left,int right,int []temp){
        if(left<right){
            int mid = (left+right)/2;
            sort(arr,left,mid,temp);//左边归并排序，使得左子序列有序
            sort(arr,mid+1,right,temp);//右边归并排序，使得右子序列有序
            merge(arr,left,mid,right,temp);//将两个有序子数组合并操作
        }
    }
    private static void merge(int[] arr,int left,int mid,int right,int[] temp){
        int i = left;//左序列指针
        int j = mid+1;//右序列指针
        int t = 0;//临时数组指针
        while (i<=mid && j<=right){
            if(arr[i]<=arr[j]){
                temp[t++] = arr[i++];
            }else {
                temp[t++] = arr[j++];
            }
        }
        while(i<=mid){//将左边剩余元素填充进temp中
            temp[t++] = arr[i++];
        }
        while(j<=right){//将右序列剩余元素填充进temp中
            temp[t++] = arr[j++];
        }
        t = 0;
        //将temp中的元素全部拷贝到原数组中
        while(left <= right){
            arr[left++] = temp[t++];
        }
    }

6.快速排序

1.算法描述
     通过一趟排序将待排记录分隔成独立的两部分，其中一部分记录的关键字均比另一部分的关键字小，则可分别对这两部分记录继续进行排序，以达到整个序列有序。时间复杂度为小于O(n^2)，空间复杂度O(n，算法不稳定，效率最高。
2.算法实现

 public static void quickSort1(int[] arr, int low, int high) {//low：0，high:len-1
    int i,j,index,tmp;
    if (low > high){
        return;
    }
    i=low;
    j=high;
    index = arr[low];
    while(i！=j){
        while (index<=arr[j] && i < j){
            j--;
        }
        while (index >= arr[i] && i<j){
            i++;
        }
        if (i<j){
            tmp=arr[j];
            arr[j]=arr[i];
            arr[i]=tmp;
        }
    }
    arr[low] = arr[i];
    arr[i] = index;
    quickSort1(arr,low,j-1);
    quickSort1(arr,j+1,high);
}

七.桶排序

1.算法描述
     通过借用一个一维数组记录每个数字出现的次数，按顺序依次输出每个数组变量的个数就达到了排序的目的。算法稳定，时间复杂度（n*k），空间复杂度（n+k。

2.算法实现

 public static void Bucket_Sort(int []array){ 
     int max = 0;
    for (int i : array) {
        if (i > max){
            int tmp = i;
            i = max;
            max = tmp;
        }
    }
    int []b= new int[max+1];
    System.out.println(max);
    for (int i = 0; i < b.length; i++) {
        b[i] = 0;
    }
    for (int i : array) {
        b[i] =b[i]+1;
    }
    for (int i = 0; i < b.length; i++) {
        for (int j = 1; j <= b[i]; j++) {
            System.out.print(i);
        }
    }
}
